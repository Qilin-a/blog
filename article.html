如何快速上手Tailwind CSS

2024年5月10日 阅读时间：5分钟 12条评论

Tailwind CSS 是一款“实用优先”的CSS框架，它不提供预设组件（如按钮、卡片），而是通过预定义的原子类（如 text-center、mt-4）让你直接在HTML中编写样式，无需额外写自定义CSS。

一、快速引入Tailwind CSS

新手无需下载本地文件，直接用CDN引入，在HTML的<head>中添加：

<script src="https://cdn.tailwindcss.com"></script>

引入后即可直接使用所有Tailwind类，无需任何配置。

二、核心使用技巧

  1. **“先试后记”**：不用死记所有类，需要时查官方文档，常用类（如flex、bg-white）会逐渐记住。
  2. **响应式设计**：直接在类名前加屏幕前缀（如md:flex表示“中等屏幕以上显示flex”），无需写媒体查询。
  3. **自定义样式**：如果需要特殊颜色/尺寸，可在<script>中配置，例如：
<script>
  tailwind.config = {
    theme: {
      extend: {
        colors: {
          mycolor: '#123456', // 自定义颜色
        },
      },
    }
  }
</script>

三、推荐工具

安装VS Code插件「Tailwind CSS IntelliSense」，可自动提示类名，大幅提升效率。
